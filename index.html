Roller coaster designer

Jonathan Bridge & Peter Froud

Project description

We will make an interface similar to RollerCoaster Tycoon, where the user will build a 3D, voxel-based roller coaster track from track pieces. They will be able to navigate using the mouse.

Original feature list

Description Priority
3D grid and editor interface Core
Track building and rendering Core
Moving roller coaster train on track High
Camera the rides the train on the track High
Supports on track Medium
Basic potential / kinetic energy simulation Medium
Track pieces that look like real track instead of a plane or ribbon Low
Load/save track to file Low
Loops, corkscrews, etc. Low
